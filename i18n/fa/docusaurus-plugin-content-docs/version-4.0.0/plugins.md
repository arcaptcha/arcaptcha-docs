# پلاگین و کتابخانه ها

ARCaptcha کتابخانه‌ها و پلاگین‌های رسمی و نگهداری شده توسط جامعه را برای فریمورک‌ها و پلتفرم‌های مختلف ارائه می‌دهد. کتابخانه‌ای را انتخاب کنید که با پشته توسعه شما بهترین سازگاری را دارد.

## کتابخانه‌های فرانت‌اند

کتابخانه‌های فرانت‌اند به شما کمک می‌کنند تا ویجت‌های ARCaptcha را در برنامه‌های وب خود ادغام کنید.

| فریمورک         | توضیحات                                                                   | لینک                                                                |
| --------------- | ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| React           | کتابخانه کامپوننت رسمی React برای ادغام ARCaptcha در برنامه‌های React     | [arcaptcha-react](https://github.com/arcaptcha/Arcaptcha-React-js)  |
| Vue             | کتابخانه کامپوننت رسمی Vue.js برای ادغام ARCaptcha در برنامه‌های Vue 2    | [arcaptcha-vue](https://github.com/arcaptcha/arcaptcha-vue)         |
| Vue3            | کتابخانه کامپوننت رسمی Vue 3 برای ادغام ARCaptcha در برنامه‌های Vue 3     | [arcaptcha-vue3](https://github.com/arcaptcha/arcaptcha-vue3)       |
| Angular         | کتابخانه کامپوننت رسمی Angular برای ادغام ARCaptcha در برنامه‌های Angular | [arcaptcha-angular](https://github.com/arcaptcha/arcaptcha-angular) |
| Flutter         | پلاگین رسمی Flutter برای ادغام ARCaptcha در برنامه‌های موبایل Flutter     | [arcaptcha-flutter](https://github.com/arcaptcha/arcaptcha-flutter) |
| Flutter (جامعه) | پلاگین Flutter نگهداری شده توسط جامعه برای ARCaptcha                      | [ar_captcha](https://github.com/fateme-shm/ar_captcha)              |

## SDK های موبایل

SDK های بومی موبایل برای برنامه‌های Android.

| SDK               | توضیحات                                                                   | لینک                                                                                      |
| ----------------- | ------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Android (بومی)    | SDK بومی Android برای ARCaptcha با پشتیبانی از چالش‌های Question و Puzzle | [arcaptcha-native-android-sdk](https://github.com/arcaptcha/arcaptcha-native-android-sdk) |
| Android (WebView) | SDK Android با استفاده از WebView برای رندر کردن ویجت‌های ARCaptcha       | [arcaptcha_android_sdk](https://github.com/arcaptcha/arcaptcha_android_sdk)               |
