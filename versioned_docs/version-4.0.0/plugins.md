# Plugins and libraries

ARCaptcha provides official and community-maintained libraries and plugins for various frameworks and platforms. Choose the library that best fits your development stack.

## Frontend Libraries

Frontend libraries help you integrate ARCaptcha widgets into your web applications.

| Framework           | Description                                                                          | Link                                                                |
| ------------------- | ------------------------------------------------------------------------------------ | ------------------------------------------------------------------- |
| React               | Official React component library for integrating ARCaptcha in React applications     | [arcaptcha-react](https://github.com/arcaptcha/Arcaptcha-React-js)  |
| Vue                 | Official Vue.js component library for integrating ARCaptcha in Vue 2 applications    | [arcaptcha-vue](https://github.com/arcaptcha/arcaptcha-vue)         |
| Vue3                | Official Vue 3 component library for integrating ARCaptcha in Vue 3 applications     | [arcaptcha-vue3](https://github.com/arcaptcha/arcaptcha-vue3)       |
| Angular             | Official Angular component library for integrating ARCaptcha in Angular applications | [arcaptcha-angular](https://github.com/arcaptcha/arcaptcha-angular) |
| Flutter             | Official Flutter plugin for integrating ARCaptcha in Flutter mobile applications     | [arcaptcha-flutter](https://github.com/arcaptcha/arcaptcha-flutter) |
| Flutter (Community) | Community-maintained Flutter plugin for ARCaptcha                                    | [ar_captcha](https://github.com/fateme-shm/ar_captcha)              |

## Mobile SDKs

Native mobile SDKs for Android applications.

| SDK               | Description                                                                      | Link                                                                                      |
| ----------------- | -------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| Android (Native)  | Native Android SDK for ARCaptcha with support for Question and Puzzle challenges | [arcaptcha-native-android-sdk](https://github.com/arcaptcha/arcaptcha-native-android-sdk) |
| Android (WebView) | Android SDK using WebView for rendering ARCaptcha widgets                        | [arcaptcha_android_sdk](https://github.com/arcaptcha/arcaptcha_android_sdk)               |

## Backend Libraries

Backend libraries help you verify ARCaptcha responses on your server.

| Language/Framework | Description                                                       | Link                                                                   |
| ------------------ | ----------------------------------------------------------------- | ---------------------------------------------------------------------- |
| Node.js            | Official Node.js library for server-side ARCaptcha verification   | [arcaptcha-nodejs](https://github.com/arcaptcha/arcaptcha-nodejs)      |
| PHP                | PHP library for server-side ARCaptcha verification                | [arcaptcha-php](https://github.com/arcaptcha/arcaptcha-php)            |
| Python             | Python library for server-side ARCaptcha verification             | [arcaptcha-python](https://github.com/evokelektrique/arcaptcha-python) |
| Ruby               | Ruby library for server-side ARCaptcha verification               | [arcaptcha-ruby](https://github.com/evokelektrique/arcaptcha-ruby)     |
| Go                 | Go library for server-side ARCaptcha verification                 | [arcaptcha-go](https://github.com/arcaptcha/arcaptcha-go)              |
| C#                 | C# library for server-side ARCaptcha verification                 | [Arcaptcha.CSharp](https://github.com/arcaptcha/Arcaptcha.CSharp)      |
| Elixir             | Elixir library for server-side ARCaptcha verification             | [arcaptcha-elixir](https://github.com/evokelektrique/arcaptcha-elixir) |
| Laravel            | Laravel package for integrating ARCaptcha in Laravel applications | [arcaptcha-laravel](https://github.com/arcaptcha/arcaptcha-laravel)    |
